# Frontend integration

## Ubicación del frontend

**NO está en este repo.** Vive en `IITA-Proyectos/iitacrm` → `AgentRuntime.jsx`.

## Features del frontend

- Inline prompt fragment editing
- Star ratings (1-5) por respuesta
- Shadow compare v3 vs v4
- Webhook monitor en tiempo real

## Deploy frontend

Siempre a `IITA-Proyectos/iitacrm` (Vercel), NO a `gviollaz/iita-system`.

Workflow: Pull → cambios → ACP desde IITA Git Panel.
